Belum terhubung
📖 Panduan Setup Supabase (5 menit)
  1. Buka supabase.com → Daftar / Login (gratis)
  2. Klik New Project → isi nama & password → Create
  3. Setelah project siap, buka SQL Editor dan jalankan query ini:
    create table kpi_data (
      id bigint generated always as identity primary key,
      data jsonb not null,
      updated_at timestamptz default now()
    );
    insert into kpi_data (data) values ('[]'::jsonb);
  4. Buka Settings → API → copy Project URL dan anon public key
  5. Paste di kolom di atas → klik Uji Koneksi & Simpan
  6. Klik Push Data ke Cloud untuk upload data lokal pertama kali
  7. ✅ Selesai! Dashboard sekarang sync otomatis setiap kali ada perubahan data
